## Getting Started
#### 1. Install
```powershell
yarn
```

#### 2. Start dev server
```powershell
yarn start:dev
```
[localhost:4000](http://localhost:4000)

```powershell
yarn firebase:dev
```
[localhost:5001](http://localhost:5001/takuma-ru-homepage/asia-northeast1/api)

#### 4. Build this App
```powershell
yarn build
```

#### 5. Deploy
```powershell
yarn firebase:deploy
```
